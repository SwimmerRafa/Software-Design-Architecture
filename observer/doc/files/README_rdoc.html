Observer Method Pattern¶ ↑

This documentation was developed as part of a programming activity for the Software Design and Architecture course (Tc3049), at the Tecnológico de Monterrey, Campus Estado de México.

Overview¶ ↑

The Observer Method is a software design pattern in which an object, called the subject, maintains a list of its dependents, called observers, and notifies them automatically of any state changes, usually by calling one of their methods.
